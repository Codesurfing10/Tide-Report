Location Widget Integration Example

How to Use

The location widget appears in the top-left corner. Try these:

  • Type a city name: San Francisco
  • Type coordinates: 37.7749, -122.4194
  • Type any place worldwide: Tokyo, Japan

Integration Code

Add these lines to your HTML:

<!-- Include CSS and JS -->
<link rel="stylesheet" href="public/location-widget.css">
<script src="public/location-widget.js"></script>

<script>
    // Initialize the widget
    createLocationWidget();
    
    // Listen for location selection
    window.addEventListener('location-selected', function(event) {
        const location = event.detail;
        console.log('Selected location:', location);
        console.log('Display name:', location.display_name);
        console.log('Latitude:', location.lat);
        console.log('Longitude:', location.lon);
        
        // Use the location data for your app
        // e.g., fetch tide data, update map, etc.
    });
</script>
